Node.js 上安裝 Hexo 並在 GitHub 建立 Blog step by step 教學-安裝

A fast, simple & powerful blog framework powered by Node.js.

Why Hexo ?

  • 使用 Hexo 建立個人 Blog 的好處有以下：
    1. 使用 Node.js 環境建立
    2. 使用 NPM 管理套件安裝
    3. 使用 Markdown 語言快速編輯，筆記、Blog 一次搞定。
    4. 文章裡面可以放上 Code 並加上 highlight.js 高亮語法顯示。
    5. 可使用 GitHub 版本管理 / 備份
    6. 最後可以直接發佈到 GitHub Page / GitHub.io

根據以上多種優點，因此建立一個 Hexo Blog Project 可以一次學習到多種工具與管理套件，因此也成為我買下 MBPR 後第一個想要玩的技術，接著就直接進入主題。
