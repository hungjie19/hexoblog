Hexo add DISQUS Comments 新增留言板

DISQUS

Hexo 本身內建 Disqus 是一款專門針對部落格、網站留言機制做統合的服務，使用者想要留言可以匿名、DISQUS會員、Facebook帳號、Twitter帳號、Google帳號進流網站留言，同時這套留言機制可以幫你統計訪客留言排行榜，另外在留言下方也可以最近被熱門討論的文章，此外也可以透過本身的分享機制把文章分享出去，並且留言機制中支援階層式(巢狀式)留言。引用：香腸炒魷魚

因此我們將在 Blog 中新增 DISQUS 留言板，步驟如下：

  1. 申請帳號
  2. 新增頁面
  3. 記下 shortname
  4. 選擇外掛可以不須理會
