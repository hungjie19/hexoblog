Hexo sitemap 提交至 Google 網站管理員 (Web Masters)

step.1

首先需先安裝 Hexo sitemap install

測試：主頁後面加上 /sitemap

Login Google 網站管理員 / webMasters

step.3

新增網站

輸入網站後 google 將進行驗證，測試成功後既可進入提交 sitemap
