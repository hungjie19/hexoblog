廖洧杰 - 前端就職技巧123

講者 PPT

  • F2E 職缺越來越多
  • 前端兩大類：
    • 前端設計師：圖片, UI/UX
      • 網頁設計師 與 前端設計師差別
      • 須懂 Git 版本控制 與 工程師合作開發
    • 前端工程師：
      • 與設計師配合開發前端組件
  • 有點雷的缺（包山包海、定義不明確）
    • 設計平台、UI/UX
    • 學過iOS..
    • 懂JAVA
  • 職缺不好的寫法
    • HTML5CSSJavaScript
    • 沒有詳細寫到深入程度
    • 請不要寫與此工作無關的相關經驗
  • 產值效益
    • 具備Responsive Mobile Layout設計經驗
    • 熟悉PhoneGap與WebAPP開發經驗
  • 工作經歷（條列式）
    • XX科技公司 - 前端工程師 (2013~2015)
      • Mobile 前端效能優化
      • IOS Hybrid APP
  • 觀察別人的履歷
    • toptal
    • Linkedin
  • 投遞履歷
    • 不要用 word 可使用 PDF
    • 可使用個人網站 or GitHub
    • 了解 HR 審核履歷流程
  • 面試須注意的細節
    • 調查公司
    • 面試過程務必確認工作內容
    • 薪水心裡要有明確價碼
    • 不懂不要裝懂
    • 公司大多喜歡有主動學習的心態
  • 千萬不要說
    • 我不會，但我願意學
  • 人力銀行
    • github jobs
    • Linkedin
    • 找一個職缺需求來訂定投入目標
