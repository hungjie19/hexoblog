目標：

  1. 讀取 csv 檔案
  2. 解析 csv 成 JSON 檔案
  3. 儲存 JSON 檔案

開始之前：

我們將使用 npm 上面的一個模組 csvtojson 來幫助我們將 csv 解析成 JSON.

    • https://www.npmjs.com/package/csvtojson
