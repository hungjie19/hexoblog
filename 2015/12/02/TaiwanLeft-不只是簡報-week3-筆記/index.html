規劃力之1 方向

疑問簡報的定義是什麼？

  • 要有圖片文？
  • 要用 PPT 做才是簡報？
  • 要有很多動畫特效？

簡報的核心關鍵：內容！

對我來說簡報就像寫作、畫圖、作曲一樣..是一種溝通形式。

竟然是溝通形式的話..你要告訴別人什麼內容才是最重要的！

溝通都會有目的性！

  • 讓人理解
  • 讓對方接受
  • 令別人內化
  • 產生行動

有目的的溝通：為了說服！

舉例有目的的溝通：

  • 簡報講師:
    • 用我的方式來做一個好的簡報
  • 政治人物:
    • 演講是為了希望選民把票投給他。
  • 賈伯斯:
    • 發表會是為了說服大家 apple 的產品是最美好的。

簡報目的：

  • 理解
  • 行動
  • 改變

有了目的還不夠，必須瞭解聽眾的對象是什麼?

簡報目的舉例：

  • 政治人物
    • 不可能對小朋友演講，因為小朋友沒有投票權。
  • 賈伯斯
    • 不可能也不會像老爺爺推薦 iPhone

聽眾目標對象：

  • 背景
  • 在意什麼 (重點)
    • 敘述簡報前，推測對方的想法
  • 影響

聽眾目標在意什麼舉例：

  • 投遞企劃履歷（壞案例）
    • 我家有爸爸媽媽一個哥哥一個姊姊兩隻狗 @#$%^&-…
    • 很顯然面試官不會因為你家有誰而入取你
    • 提供別人不需要的資訊
  • 投遞企劃履歷（參考）
    • 提出類似經驗
    • 提出相關經驗
    • 展現熱誠與企圖心
    • 證明發展淺力
    • 提供別人需要的資訊
  • 提出對方不知道，應該在意的事情（轉個彎思考）
    • 飲料店需要手腳俐落，身手敏捷的員工
    • 自己手腳不是很快不太敏捷
    • 提出記憶力很好可以節省時間做好所有訂單
    • 因為記憶力好可以解決出餐速度問題 ＝ 敏捷
    • 展現機智，證明自己。

簡報是個旅程

  • 對象是起點
  • 目的是終點
  • 必須帶領聽眾從起點走到終點
  • 釐清起點與終點才可以勾勒出簡報的方向
  • 對象不同陳述內容也不同

假設你要推廣游泳運動

  • 對象：在意自己身材
    • 強調游泳可以雕朔身材看起來更均勻
  • 對象：準備當兵
    • 強調學會游泳在部隊中的好處：可以提早放假，很多時間可以休息
  • 假設你要介紹：霧霾
    • 每天都暴露在 PM 2.5 之下
    • 對下一代有什麼影響
  • 掌握
    • 創造聽眾的在意點
    • 尋找共同的在意點

對象是起點
目的是終點
掌握了三點就掌握了說服的基本原則
簡報的旅程才算是正式的踏出了第一步

規劃力之2 架構

  • 太多重點
  • 網狀脈絡 -> 線性脈絡
  • 釐清線性脈絡
    • 根據時間發展
    • 從巨觀到微觀
    • 論證分析事件

小紅帽故事舉例：

  • 時間軸
    1. 小紅帽探望奶奶
    2. 野狼吞到小紅帽
    3. 獵人救了小紅帽
    4. 野狼死了
  • 時間軸變形
    1. 野狼死了
      • 聽眾疑問為什麼？誰殺的？
    2. 小紅帽探望奶奶
      • 野狼是被他們兩個其中之一殺死的嗎？
    3. 野狼吞到小紅帽
      • 原來野狼是壞人！奶奶怎麼救小紅帽的？或者有別人搭救？
    4. 獵人救了小紅帽
      • 原來是遇到獵人出手相救！！

變形的好處可以吧主角是不是野狼大野狼
一開始既破題指出大野狼死掉了！
觀眾也許會疑問說，為什麼大野狼為什麼會死？
接著時間轉移到小紅帽為什麼出門因為要去探望奶奶

舉例時間軸範例電影：超時空攔截（2014）

從巨觀到微觀

  1. 趨勢全國 70% 都在使用智慧手機
  2. 在分地區比例
  3. 再分年齡比例

介紹拍賣平台

  • 買家與賣家
  • 物流與金流

從微觀到巨觀

  1. 20 年傳統夜市人潮流失
  2. 隔壁小夜市人潮卻暴增
  3. 發現小夜市兩條街口外有捷運出口造成人潮改變
  4. 因為人潮改變進而改變市場結構。

從微觀案例觀察它的變化，進而發現問題，再從問題看到更宏觀的角度，開拓聽眾分析事情角度的視野。

論證分析事件

  • 目的：改變公司加班制度
  • 對象：老闆
  • 方向：利潤

論證分析舉例：

新任主管發現很多人加班，但公司績效卻不如預期？

舉出：日本同業某公司營收暴增，卻沒有做任何新產品或合併！（那他做了什麼？）

老闆宣布修改取消加班費機制政策，改為績效獎勵機制！

半年後發現，上班的時候每個人充滿幹勁把效率發揮到最高，鼓勵做得好、做得多的員工，而不是工作做很久一直加班的員工。

再舉出：有個類似的公司，有一個人才出走了，因為老闆只看到加班最多人認為他最努力..。

實作小技巧：
再開始製作PPT的時候只寫上標題重點，開始排序，直到釐清脈絡在開始詳細每個章節。

規劃力之3 內容

穹頂之下 - 柴靜

  • 目的：呼籲聽眾防治污染
  • 架構：我們每天都接觸 PM2.5 對下一代有負面影響
  • 內容：用自己角度陳述

懷疑自己小孩患有良性腫瘤

史丹佛大學演講 - 賈伯斯

  • 故事 - 創辦蘋果電腦 被解雇
  • 啟示 - 相信這是自己熱愛的事
  • 行動 - 進了兩間公司最後返回蘋果
  • 成果 - 帶領頻果電腦創造新高峰
  • 啟示 - 呼籲聽眾不要放棄熱愛的事

規劃力之4 呈現

簡報畫面，聽眾感知

簡報內有什麼重點，需要被知道的表現方式

用聽的無法理解

如：數據，畫面

  • who/what :誰、大小
  • how much: 數量：圖表
  • where: 位置：關聯、結構、組合、對立
  • when: 時間性：序列、週期性
  • how: 步驟性：如果
  • why: 分析性：為什麼有這樣的結果

講師簡報呈現案例分享

  • 時間順序
    • 介紹可汗學院
  • 巨觀到微觀
    • 當新聞不再新
      • 引用分析：現在人的獲取資訊的管道與以前不同，發現趨勢
      • 細部分析個案族群獲取資訊的管道
    • 數位掀浪
      • Borders
  • 論證形式
    • 單字強力教 APP
    • 單字學習的邏輯
    • 分析現有 app 學習流程
    • 得出學習結論
