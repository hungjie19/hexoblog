使用 hackmd online editor 與 markdown 語法製作工程師簡報

HackMD 投影片語法介紹：

章節與小節語法

按鍵與方向 定義 分頁符號
→ 章節 - - -
↓ 小節 - - - -

範例：

假設我們要介紹一本書從首頁～第二章它的結構如下

  • 首頁
  • 第一章
    • 1.1 節
    • 1.2 節
  • 第二章

語法範例：

小提醒

  • 分頁符號最好前後加上空行，才不會有語法沒有解析到的情形
  • 分頁符號解析成功在編輯器內會變成藍色
  • 按下 esc 可以看到簡報結構

調整投影片轉場動畫

Is the transition syntax

轉場動畫其他效果：

效果名稱 效果
none 無
fade 淡入淡出
slide 向左平移
convex 向內旋轉
concave 向外旋轉
zoom 向外放大

並且可以分別為投影片 進/出場，加上後綴符號設定：

  • 進場：-in
  • 出場：-out
