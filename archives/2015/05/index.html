安裝hexo2.8.3版本

如果安裝過程出現權限錯誤，可在命令前加上sudo

npm install hexo@v2.8.3 -g（安裝2.8.3版本）

npm uninstall hexo（解除安裝hexo）

建置hexo存放資料夾

mkdir [filename]

cd [filename]

初始化

hexo init

npm install

如果無法安裝請嘗試以下命令
npm install -g -unsafe-perm hexo

hexo server 啟動失敗 issue

sudo npm install hexo-renderer-marked@0.1 --save
sudo npm install hexo-renderer-stylus@0.1 --save
